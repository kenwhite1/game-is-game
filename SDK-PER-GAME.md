# Game is Game — SDK интеграция по играм

> Автогенерируется из `shared/achievements.ts` (`scripts/gen-sdk-doc.ts`). Не редактируй руками.

Каждая игра при окончании матча вызывает `ggReport(HUB_URL, launchToken, report)` из `shared/sdk.ts`.
Токен запуска игра получает из `startapp`-параметра. Хаб сам считает награды/прогресс/ранг —
игре нужен ОДИН вызов. Повтор с тем же `idempotencyKey` идемпотентен.

## Общий контракт (`MatchReport`)

```ts
await ggReport(HUB_URL, launchToken, {
  idempotencyKey: 'game:match:<matchId>:<userId>', // уникально на матч/игрока
  result: 'win' | 'loss' | 'draw' | 'finish',      // finish — для раннеров/пасьянсов
  players, humanPlayers,   // размер лобби и сколько ЖИВЫХ людей (важно для соц./ранга)
  mode: 'solo' | 'multi' | 'friends',
  durationSec, score,       // опционально
  stats: { /* см. ниже по игре */ },
})
```

**Автоматически (без `stats`)** хаб считает из полей отчёта:
- «Первая победа» / «Победитель» / прогресс — из `result` (`win`/`finish` = успех).
- «Завсегдатай» — из числа матчей.
- «Против людей» — из побед при `humanPlayers ≥ 2`.
- «Знаток режимов» — из разных `mode`.
- «Мастер игры» — открывается сам, когда взяты все прочие достижения игры.

**Только флаги/числа в `stats`** нужны для «фирменных»/скилловых достижений ниже —
булев `true` (когда игрок совершил приём) или число (накопление). Слать их нужно только
в матче, где приём случился; в остальных — не слать.
