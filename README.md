# Game is Game 🎮

Один бот, чтобы закончить со всеми ботами. **Game is Game** это лаунчер: уютная
комната, в которой собраны все наши Telegram-игры. Открываешь приложение, видишь
меню и нажимаешь на игру. Она запускается сразу, без чатов и лишних шагов.

> **Запущено:** бот [@gameisgamebot](https://t.me/gameisgamebot) ·
> приложение https://game-is-game-production.up.railway.app

В меню весь наш каталог: 37 игр в семи категориях (карты, настольные,
компания, слова, аркады, головоломки, стратегии), с поиском, фильтрами,
избранным, оценками 👍/👎 и полкой «Популярное» по реальным запускам.
Флагманы:

- 🎴 **Однушечка** · карты по цвету и числу, в духе «Уно».
- 🐊 **Крокоша** · игра в ассоциации, цифровой «Крокодил».
- 🌙 **Секрет ночи** · тайные роли, блеф и интуиция, в духе «Мафии».
- 🐾 **Шарик** · питомец, о котором заботишься, немного как тамагочи.

Игры с контентом 18+ (Картишки, Слабо?) помечены значком прямо на плитке.
Полный список живёт в `shared/games.ts`.

Дизайн и тон такие же, как у наших игр: тёплый кремовый фон, шрифт Nunito,
глянцевые плитки и мягкий ламповый свет.

## Как устроено

Это монорепозиторий с тремя рабочими пространствами, как у остальных игр:

```
shared/   общий каталог игр (единый источник правды) и типы
server/   Hono-сервер: бот, авторизация, мини-API, раздача SPA
app/      React + Vite: экран-лаунчер с меню игр
```

### Связь между играми

Каждая плитка открывает Mini App нужной игры по прямой ссылке вида
`https://t.me/<бот_игры>?startapp=gg`. Внутри Telegram такая ссылка запускает игру
сразу, в полноэкранном режиме, без перехода в чат. Так лаунчер собирает все
игры под одной кнопкой, не ломая их отдельные деплои.

Ссылки берутся из `shared/games.ts`. Если у какой-то игры другой `@username`
или нужен полностью свой URL, переопредели его переменной окружения
(`UNO_BOT`, `UNO_LINK` и т.д., см. `.env.example`), менять код не нужно.

> Чтобы `?startapp` открывал игру сразу, у каждого бота-игры должно быть
> включено главное мини-приложение (BotFather → Bot Settings → Configure Mini App).

### Команды бота

| Команда | Что делает |
|---|---|
| `/start` | Открыть меню игр |
| `/play` | Открыть меню игр |
| `/games` | Список игр со ссылками |
| `/invite` | Личная ссылка-приглашение (+250 Game за друга) |
| `/about` | Об этом приложении |
| `/help` | Помощь |

Рядом с полем ввода также появляется кнопка меню «Игры», открывающая лаунчер.

### Рефералка

У каждого игрока есть ссылка `t.me/<бот>?startapp=ref_<КОД_ДРУГА>` (кнопка
«Пригласить друга» на вкладке «Друзья» или команда `/invite`). Новый игрок,
зашедший по ней, приносит пригласившему +250 Game, сам получает +150 Game, и
они сразу становятся друзьями. Засчитывается только первый вход нового
аккаунта (`start_param` подписан Telegram, подделать нельзя); уже играющих
ссылка просто добавляет в друзья. За приглашения выдаются значки «Зазывала»
(1 друг) и «Амбассадор» (5 друзей) с эксклюзивной косметикой.

## Локальный запуск

```bash
npm install
npm run dev:server   # сервер на :3000 (DEV MODE без BOT_TOKEN)
npm run dev:app      # приложение на :5174 с прокси /api на :3000
```

Без `BOT_TOKEN` сервер работает в DEV MODE: авторизация заглушена под одного
локального игрока, бот не активен. Идеально для проверки меню.

## Проверки

```bash
npm run typecheck    # сервер + приложение
npm run build        # сборка SPA в app/dist
```

Health-check: `GET /api/health` → `{"ok":true}`.
